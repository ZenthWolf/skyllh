    "## Factorized Flux Function"
   ]
  },
  {
   "cell_type": "raw",
   "id": "98ebd803",
   "metadata": {
    "raw_mimetype": "text/restructuredtext"
   },
   "source": [
    "Usually the flux function can be split into a spatial, energy, and time profile with an overall normalization constant in differential flux unit, i.e. :math:`\\mathrm{area}^{-1} \\mathrm{solid-angle}^{-1} \\mathrm{energy}^{-1} \\mathrm{time}^{-1}`. Hence, SkyLLH provides the class :py:class:`~skyllh.core.flux_model.FactorizedFluxModel`, which describes a differential flux function as the product of individual flux profiles:\n",
    "\n",
    ".. math::\n",
    "\n",
    "    \\Phi(\\alpha,\\delta,E,t | \\vec{p}_\\mathrm{s}) =\n",
    "            \\Phi_0\n",
    "            \\Psi(\\alpha,\\delta|\\vec{p}_\\mathrm{s})\n",
    "            \\epsilon(E|\\vec{p}_\\mathrm{s})\n",
    "            T(t|\\vec{p}_\\mathrm{s})\n",
    "\n",
    "The abstract base class for any flux profile is :py:class:`~skyllh.core.flux_model.FluxProfile`, which is derived from :py:class:`~skyllh.core.math.MathFunction`.\n",
    "\n",
    "The abstract base class for a spatial, energy, and time flux profile is :py:class:`~skyllh.core.flux_model.SpatialFluxProfile`, :py:class:`~skyllh.core.flux_model.EnergyFluxProfile`, and :py:class:`~skyllh.core.flux_model.TimeFluxProfile`, respectively, and are derived from :py:class:`~skyllh.core.flux_model.FluxProfile`."
   ]
